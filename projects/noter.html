NOTER

The simple note-taking app

What is Noter?

Noter is a simple, easy to use note making app - not to be confused with a text editor like Notepad. You see, Noter allows you to store all of your notes within the app itself, so there is no need to look for your notes from your file system, keeping it clean and tidy.

Already have notes as .txt files?

That's OK! Noter allows you to import any file and will convert it to a native Noter note. From then on, you can access that note from within Noter!
